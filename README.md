**EBVkrsDesktop** in **PySide6** and **QML**
EBVkrsDesktop - ui for an application for assessing the phenotypic (genetic) breeding value of cattle, written in PySide6 and Qml6.7
**Requirements:**
- Python 3.12.3
- PySide6
- Qml6.7
